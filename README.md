Welcome to my **CodeAlpha** internship repository. This workspace serves as a centralized portfolio for the Python programming projects I developed during the program. Each application was built to solve a specific logical challenge, ranging from interactive game development to financial data tracking and workflow automation.

## 📂 Featured Projects

### 1. 📈 Stock Portfolio Tracker
A financial utility application designed to help users track and manage their stock investments.
* **Core Logic:** Data structure manipulation, mathematical calculations for profit/loss, and user input validation.
* **Key Skills:** Data handling, continuous state management.

### 2. 🤖 Basic Chatbot
An interactive, text-based conversational agent capable of responding to user inputs based on predefined rules and natural language patterns.
* **Core Logic:** String parsing, conditional routing, and pattern matching.
* **Key Skills:** Logic flow, user experience (UX) in the command line.

### 3. ⚙️ Task Automation Scripts
A suite of Python scripts built to eliminate repetitive digital tasks, improving overall workflow efficiency. 
* **Core Logic:** File system operations (OS module), batch processing, and exception handling.
* **Key Skills:** Practical scripting, system administration basics.

### 4. 🎮 Hangman Game
A digital recreation of the classic word-guessing game, built to demonstrate core programming fundamentals.
* **Core Logic:** While-loops, list iterations, string masking, and game-state tracking (win/loss conditions).
* **Key Skills:** Algorithmic thinking, control flow.
